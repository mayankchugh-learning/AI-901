## Questions

### 1. Which of these statements accurately defines AI information extraction?

- Using SQL code to query a database.
- Copying files from cloud storage to local disks.
- **Correct:** Analyzing unstructured content to identify and extract relevant fields and values.

### 2. How is OCR used in the information extraction process?

- Online Content Retrieval (OCR) is used to copy cloud data to a local device for processing.
- **Correct:** Optical character recognition (OCR) is used to convert images of text into machine-readable text data.
- Open Conversion Routine (OCR) is a method for converting JSON data into images.

### 3. How does generative AI enhance the data extraction process?

- **Correct:** By using semantic language models to accurately match extracted values to data fields.
- By manually coding extraction rules for each document type.
- By generating new documents from scratch.
